# Security

Oikoscope publishes open datasets and research-oriented tooling. If you discover a security issue (for example, accidental inclusion of private credentials or personal data in the repository), please report it responsibly.

**Please do not** open a public issue for undisclosed security problems.

Contact maintainers via the [CHAOSS Data Science Working Group](https://chaoss.community/inside-the-chaoss-data-science-working-group/) on Slack (**#data-science**), or reach out to repository maintainers listed in [`README.md`](README.md).

We will acknowledge receipt and coordinate an appropriate fix and disclosure timeline.
