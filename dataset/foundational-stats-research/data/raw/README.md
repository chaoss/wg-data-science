# Raw data (moved)

**Canonical dataset:** Apache project metadata now lives only in **`oikoscope/data/raw/apache.json`** in this repository.

Contributions should update that file and follow the [Oikoscope contributing guide](../../../oikoscope/CONTRIBUTING.md) and the [community tracker](https://docs.google.com/spreadsheets/d/1rWW773oOudiM5xpfroIjEiQazOr42YNCfGWISnR0Tk4/edit?usp=sharing).

This folder is kept so existing relative links to `data/raw/` still resolve to an explanation instead of an silently outdated copy.
